Sidebar

How do I set up the Sidebar?​

Open your Config Panel.

Click Main Settings

Then, click Sidebar

Add a Sidebar. I like to call it Main or <server name> Main

Add a line. You can add any of it and make it animated.

How do I enable the sidebar​

Go to the sidebar settings and click Settings. Then, enable the sidebar

How do I make it animated?​

Making your sidebar animated is simple. Just add more frames!
